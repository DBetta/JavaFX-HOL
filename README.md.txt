Unleash the Power of JavaFX
=============


In this HOL you will have the opportunity to get your hands on real exercises, to learn and play with some of these latest features of JavaFX. We will be using NetBeans 7.1  and JavaFX Scene Builder 1.0 along the way. So, lets get started and have some fun.

Exercises
=============

* A little bit of everything: JavaFX, NetBeans, Scene Builder, and FXML
* Styling your application using CSS
* JavaFX, JavaScript, WebView and HTML5
*  Canvas API
